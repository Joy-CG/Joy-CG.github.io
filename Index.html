Project 01

PassVault 2.0

Password Manager · Cryptography · Desktop App

View on GitHub ↗

A Python desktop password manager built on strong cryptographic foundations. Uses Argon2id for key derivation and AES-128-CBC for encryption. Integrates the HaveIBeenPwned k-anonymity API for breach checking without ever transmitting the actual password, and includes a cryptographically secure password generator. The master password is never stored — an incorrect password simply results in failed decryption.

Python Argon2id AES-128-CBC HaveIBeenPwned API Cryptography Desktop App

Project 02

LogSentinel

Log Analyzer · SIEM Dashboard · Threat Detection

View on GitHub ↗

A Python-based log analyzer for detecting brute force, SQL injection, XSS, and path traversal across Apache, syslog, and Windows Event logs. Features a SIEM-style dashboard with charts, GeoIP lookup, IP whitelist/blacklist management, live log monitoring via tail -f, email alerting, and CSV/TXT export for reporting.

LogSentinel 2.0
